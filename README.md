
# Your Palette

The propose of this project is developing python app to get color palette from your favorite app.

Started on 2018-05.

## Directory structure

* Dockerfile
* LICENSE
* README.md
* app
  * DASH.png
  * app.py
  * app_test.py
  * identidock.png
  * outfile.png
  * static
    * st25.css
* cmd.sh
* docker-compose.yml


## Usage

Just uploading your favorite image in upload component.
Then you will get image palette.
